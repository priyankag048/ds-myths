# Data Structure Myths Series


Welcome to the Data Structure Myths GitHub repository. This repo contains reproducible experiments for a blog series that explores common assumptions about data structures and their real-world behavior.

## Repository Structure

- Each myth-0X folder contains experiments for a specific myth.

- Inside each myth folder, code is organized by language.

- Optional utils contains helper scripts for timing and benchmarking.

## Getting Started

Clone the repo: git clone 

Navigate to a myth folder to run experiments in your preferred language.

## Prerequisites

Make sure you have the following installed:

Java: JDK 17 or later

Go: Go 1.25 or later

Python: Python 3.9 or later

Node.js: Node 18 or later

All commands below are run from the repository root.
